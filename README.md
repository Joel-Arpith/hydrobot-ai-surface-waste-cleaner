HydroBot is an AI-powered, solar-driven autonomous surface water waste collection robot designed for lakes, ponds, and other small community water bodies.

## What HydroBot Does
- Detects floating surface waste using YOLOv8
- Navigates autonomously toward detected waste
- Collects waste using an onboard intake mechanism
- Operates on a solar + battery hybrid power system
- Logs waste count, GPS location, and sensor data to an IoT dashboard
